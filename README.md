# namrata-docs

Research documentation for Namrata's projects.

## Research Projects

### 1. Unified Size App/AI
A unified mobile application integrating AI-powered body measurement and size recommendation using computer vision and machine learning.

### 2. Mall Measuring Kiosks
Self-service kiosks deployed in retail malls for automated body measurement, enabling customers to get accurate sizing without fitting rooms.

### 3. NFC Size Scanners
Near-field communication (NFC) enabled scanning devices that capture body measurements via NFC tags embedded in clothing or accessories for seamless size matching.
