# namrata-docs

Research documentation for Namrata's projects.

---

## Abstract

This repository documents research projects focused on automated anthropometric measurement and intelligent size recommendation systems. The research encompasses three interconnected systems: a unified mobile application leveraging computer vision and machine learning for body measurement estimation, self-service kiosk deployments for retail environments, and NFC-based measurement capture technologies. These projects collectively address the challenge of accurate, accessible body sizing across diverse retail contexts.

---

## Research Projects

### 1. Unified Size App/AI

**Description:** A unified mobile application integrating AI-powered body measurement and size recommendation using computer vision and machine learning.

**Technical Approach:**
- Computer vision pipelines for landmark detection and body dimension extraction
- Machine learning models for size prediction across garment categories
- Mobile-optimized inference for real-time measurement estimation

**Applications:**
- Consumer-facing size recommendation
- E-commerce personalization
- Virtual fitting assistance

---

### 2. Mall Measuring Kiosks

**Description:** Self-service kiosks deployed in retail malls for automated body measurement, enabling customers to get accurate sizing without fitting rooms.

**Technical Approach:**
- Structured light or stereo vision depth estimation
- Privacy-preserving measurement capture protocols
- Multi-point anthropometric landmark detection
- Integration with retail point-of-sale systems

**Applications:**
- In-store customer sizing
- Retail floor deployment
- Size chart generation for regional populations

---

### 3. NFC Size Scanners

**Description:** Near-field communication (NFC) enabled scanning devices that capture body measurements via NFC tags embedded in clothing or accessories for seamless size matching.

**Technical Approach:**
- NFC tag data retrieval and parsing
- Body measurement association and reconciliation
- Cross-device size recommendation synthesis

**Applications:**
- Smart garment interaction
- Accessories-based measurement capture
- Seamless omnichannel sizing

---
